## Introduction

The Python SDK is available on [PyPI](https://pypi.org/project/skribble-sdk/).

You can install the SDK using pip:

```bash
pip install skribble-sdk
```

<Warning>This SDK is not maintained or published by Skribble. It is an unofficial open source SDK.</Warning>

## Initialize the SDK

```python
import skribble

# Replace with your actual API credentials
USERNAME: str = "api_xxxxx"
API_KEY: str = "xxxxx"

# Initialize the SDK
skribble.init(USERNAME, API_KEY)
```

## Optional: Get access token
There might be cases where you want to get the access token for the API. You can do this by using the `init` function with the `access_token` parameter.

```python
import skribble

# Replace with your actual API credentials
USERNAME: str = "api_xxxxx"
API_KEY: str = "xxxxx"

# Initialize the SDK
access_token = skribble.init(USERNAME, API_KEY)

# Get access token
print(access_token)

# Reinitialize the SDK with the access token
skribble.init(access_token=access_token)
```
