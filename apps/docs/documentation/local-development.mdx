## Introduction

Skribble SDK is set up in a monorepo (via [Turborepo](https://turbo.build/repo)) and is fully open-source.
Here's the monorepo structure:

```
apps
├── docs
├── py-sdk
├── py-sdk-examples
├── ts-sdk
packages
├── eslint-config
├── typescript-config
```

The `apps` directory contains the code for:

- `py-sdk`: The Python SDK
- `py-sdk-examples`: The Python SDK examples (not part of the SDK, but useful for local development)
- `ts-sdk`: The Typescript SDK (coming soon)
- `docs`: The documentation site (deployed on Mintlify)

The `packages` directory contains the code for:

- `eslint-config`: ESLint configurations for the codebase. (Will be used in future SDKs) 
- `typescript-config`: TypeScript configurations for the codebase. (Will be used in future SDKs) 

## Running the SDKs
### Step 1: Local setup

<Steps>
  <Step title="Clone the repo">
    Clone the [Skribble SDK repo](https://github.com/LeEricCH/skribble-sdk).

    ```bash
    git clone https://github.com/https://github.com/LeEricCH/skribble-sdk.git app-name
    ```    

  </Step>
  <Step title="Install dependencies">    
    Change to the root directory of the cloned repository and install the dependencies using the following command:

    ```bash
    cd app-name
    pnpm install
    ```      
    Install also dependencies in editable mode for the SDKs:
    ```bash
    # Uninstall the SDK if already installed
    pip uninstall skribble-sdk
    # Install the SDK in editable mode
    pip install -e apps/py-sdk
    ```
  </Step>    

</Steps>


### Step 2: Test the SDKs

```bash root
pnpm run test
```

### Step 3: Run the examples

```bash root
pnpm run run-examples
```
