## Introduction

The Python SDK is available on [PyPI](https://pypi.org/project/skribble-sdk/).

You can install the SDK using pip:

```bash
pip install skribble-sdk
```

## Initialize the SDK

```python
import skribble

# Replace with your actual API credentials
USERNAME: str = "api_xxxxx"
API_KEY: str = "xxxxx"

# Initialize the SDK
skribble.init(USERNAME, API_KEY)
```
