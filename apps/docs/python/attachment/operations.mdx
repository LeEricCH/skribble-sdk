## Attachment Operations

### Adding Attachments

To add multiple attachments to an existing signature request:

```python
attachments = [
    {"filename": "doc1.pdf", "content_type": "application/pdf", "content": "base64_encoded_content_1"},
    {"filename": "doc2.pdf", "content_type": "application/pdf", "content": "base64_encoded_content_2"}
]
attachment_response = skribble.attachment.add(signature_request_id, attachments)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to add the attachments to
</ParamField>

<ParamField path="attachments" type="array" required>
  List of dictionaries containing attachment information
  <Expandable title="attachments[]">
    <ParamField path="filename" type="string" required>
      Name of the attachment
    </ParamField>
    <ParamField path="content_type" type="string" required>
      Content type of the attachment
    </ParamField>
    <ParamField path="content" type="string" required>
      Base64 encoded bytes of the attachment
    </ParamField>
  </Expandable>
</ParamField>

### Getting an Attachment

To download a specific attached file from a signature request:

```python
attachment_content = skribble.attachment.get(signature_request_id, attachment_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to get the attachment from
</ParamField>

<ParamField path="attachment_id" type="string" required>
  ID of the attachment to get, you can get the ID from the signature request response
</ParamField>

### Deleting an Attachment

To remove an attachment from a signature request:

```python
skribble.attachment.delete(signature_request_id, attachment_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to delete the attachment from
</ParamField>

<ParamField path="attachment_id" type="string" required>
  ID of the attachment to delete, you can get the ID from the signature request response
</ParamField>

## Listing Attachments

To list all attachments for a signature request:

```python
attachments = skribble.attachment.list(signature_request_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to list the attachments for
</ParamField>
