## Request Operations

To create a signature request, use the following code:

```
skribble.signature_request.create(signature_request)
```

### Getting a Signature Request

To get details of a specific signature request:

```python
skribble.signature_request.get(signature_request_id)
```
<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to get
</ParamField>

### Listing Signature Requests

To list signature requests:

```python
skribble.signature_request.list(limit=5)
```

<ParamField path="limit" type="int">
  Limit the number of signature requests to return
</ParamField>

### Deleting a Signature Request

To delete a signature request:

```python
delete_response = skribble.signature_request.delete(signature_request_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to delete
</ParamField>

### Withdrawing a Signature Request

To withdraw a signature request with a message:

```python
withdraw_response = skribble.signature_request.withdraw(signature_request_id, message="Withdrawing for testing purposes")
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to withdraw
</ParamField>

<ParamField path="message" type="string">
  Message to send to the signers
</ParamField>


## Signer Operations

### Adding a New Signer

To add a new signer to an existing signature request:

```python
add_signer_response = skribble.signature_request.add_signer(
    signature_request_id,
    email="newsigner@example.com",
    first_name="New",
    last_name="Signer",
    mobile_number="1234567890",
    language="en"
)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to add the signer to
</ParamField>

<ParamField path="email" type="string" required>
  Email address of the signer
</ParamField>

<ParamField path="first_name" type="string">
  First name of the signer
</ParamField>

<ParamField path="last_name" type="string">
  Last name of the signer
</ParamField>

<ParamField path="mobile_number" type="string">
  Mobile number of the signer
</ParamField>

<ParamField path="language" type="string">
  Language of the signer
</ParamField>


### Removing a Signer

To remove a signer from an existing signature request:

```python
signature_request = skribble.signature_request.get(signature_request_id)

signer_to_remove = signature_request['signatures'][0] # This is the first signer in the list

skribble.signature_request.remove_signer(signature_request_id, signer_to_remove['sid'])
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to remove the signer from
</ParamField>

<ParamField path="signer_id" type="string" required>
  SID of the signer to remove
</ParamField>

## Attachment Operations

### Adding Attachments

To add multiple attachments to an existing signature request:

```python
attachments = [
    {"filename": "doc1.pdf", "content_type": "application/pdf", "content": "base64_encoded_content_1"},
    {"filename": "doc2.pdf", "content_type": "application/pdf", "content": "base64_encoded_content_2"}
]
attachment_response = skribble.attachment.add(signature_request_id, attachments)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to add the attachments to
</ParamField>

<ParamField path="attachments" type="array" required>
  List of dictionaries containing attachment information
</ParamField>

<ParamField path="attachments[].filename" type="string" required>
  Name of the attachment
</ParamField>

<ParamField path="attachments[].content_type" type="string" required>
  Content type of the attachment
</ParamField>

<ParamField path="attachments[].content" type="string" required>
  Base64 encoded bytes of the attachment
</ParamField>

### Getting an Attachment

To download a specific attached file from a signature request:

```python
attachment_content = skribble.attachment.get(signature_request_id, attachment_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to get the attachment from
</ParamField>

<ParamField path="attachment_id" type="string" required>
  ID of the attachment to get, you can get the ID from the signature request response
</ParamField>

### Deleting an Attachment

To remove an attachment from a signature request:

```python
skribble.attachment.delete(signature_request_id, attachment_id)
```

<ParamField path="signature_request_id" type="string" required>
  ID of the signature request to delete the attachment from
</ParamField>

<ParamField path="attachment_id" type="string" required>
  ID of the attachment to delete, you can get the ID from the signature request response
</ParamField>
